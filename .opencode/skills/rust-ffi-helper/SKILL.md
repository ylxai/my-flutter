## What I do

- Generate Rust FFI bindings for Flutter
- Debug common Rust-Flutter integration issues
- Optimize performance of Rust functions called from Flutter
- Maintain consistency between Dart and Rust interfaces

## When to use me

Use this when working with the Rust backend components of your Flutter app.
